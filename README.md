# ♛ 8 Queens AI Game

An interactive and visually enhanced implementation of the classic **8 Queens Problem**.

## 🚀 Features

* 🎮 Interactive chessboard UI
* 🔊 Meme-style error feedback (FAHHH sound)
* 💥 Screen shake + red flash on invalid moves
* ⏱ Timer & move tracking
* 🔁 Undo / Redo system
* 🎉 Win detection
* 🧠 Constraint-based validation (AI logic)

## 🧠 Concept

The 8 Queens problem is a classic constraint satisfaction problem where 8 queens must be placed on a chessboard such that no two queens attack each other.

## 🛠 Tech Stack

* React.js
* JavaScript (ES6)
* CSS (inline styling)

## ▶️ Run Locally

```bash
npm install
npm start
```
